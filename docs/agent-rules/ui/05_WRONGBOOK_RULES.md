# 05_WRONGBOOK_RULES — 错题本规则

&gt; 依赖：02_UI_RULES、06_NAVIGATION_RULES | 最后更新：2026-05-31
&gt; 实现 WrongPage / 错题卡片 时必须读本文件

---

## 1. 错题状态四态

| 状态 | 左侧色条（4px）| 适用条件 |
|------|--------------|---------|
| 未复习 | danger-500 `#EF4444` | 做错后未查看 |
| 已查看 | warning-500 `#F59E0B` | 已查看解析未重练 |
| 重练中 | primary-500 `#3B82F6` | 正在重做 |
| 已掌握 | success-500 `#10B981` | 7天内连续答对 ≥ 2 次 |

多次错误（≥ 3次）：danger-700 + 右上角数字角标（红色圆形 badge）

---

## 2. 卡片结构

```
┌─ 4px色条 ──────────────────────────────────────┐
│  [年份][省份][难度⭐][知识点标签]  [错误次数] │
│  题目摘要（≤ 60字，超出截断显示 ...）          │
│  ──────────────────────────────────────────    │
│  最近作答：YYYY-MM-DD   知识点：模块 &gt; 子概念 │
└────────────────────────────────────────────────┘
```

- 卡片：`bg-white rounded-xl shadow-md`
- 难度：1-5星，accent-400 填充色
- 知识点标签：最多显示 2 个，超出 +N 折叠

---

## 3. 交互规范

| 操作 | 行为 |
|------|------|
| 悬停卡片 | `translateY(-2px)` + shadow-lg，卡片 hover 时长见 `03_MOTION_RULES §2`（反馈类 `duration.normal`，ease-out） |
| 点击卡片 | 跳转 `/analysis/:id?from=wrongbook`（当前标签页，见 06_NAVIGATION_RULES §4） |
| 已掌握卡片 | 默认折叠至列表底部，点击"显示已掌握"展开 |
| 长按/右键 | 显示上下文菜单：标记已掌握 / 删除 / 添加笔记 |

---

## 4. 筛选与排序

筛选 chip 多选（不互斥）：
- 按模块：力学 / 电磁学 / 热学 / 光学 / 原子物理
- 按难度：1-5星
- 按状态：未复习 / 已查看 / 已掌握

排序（单选）：
- 最近错误（默认）
- 错误次数多→少
- 难度高→低

筛选栏样式：chip 组件，选中态 primary-600 背景白字，未选中 neutral-100 背景 neutral-600 字

---

## 5. 数据规范

- 错题数据唯一来源：`useWrongStore`（Zustand）
- 持久化：IndexedDB via `storage.setDB`
- 禁止 WrongPage 直接读写 IndexedDB（必须通过 useWrongStore）
- 连续答对判定：同一题 `correctStreak >= 2` 且最近两次答对间隔 ≤ 7天 时触发已掌握状态
- 超过 7 天间隔的答对不计入连续，`correctStreak` 重置为 1
- 笔记字数上限：200字
- 首次掌握提示：触发已掌握时显示「已掌握」toast（success-500 色，3 秒后自动消失，无感叹号）
- 7 天后复习提醒：已掌握超过 7 天的题目，右上角数字角标变为 🔄 图标（neutral-400 灰色，非 danger 红色），提示用户重新测试

---

## 6. 统计面板（页面顶部）

```
总错题数: N  |  未掌握: N  |  本周新增: N  |  本周掌握: N
```

- 数值用 mono 字体，neutral-800
- 未掌握数字：danger-600（有压迫感，驱动复习）
- 本周掌握：success-600（正向激励）

---

## 7. 禁止项

- 禁止直接操作 IndexedDB（必须走 useWrongStore）
- 禁止删除未确认（需二次确认 Modal）
- 禁止已掌握卡片默认显示在列表顶部
- 禁止筛选结果为空时崩溃（显示空态，文案见下方）

---

## 8. 空态设计

全空态（无任何错题）和筛选空态（无匹配结果）必须区分，且文案风格符合 02_UI §1「精密学习工具」气质——克制、中性、无感叹号。

### 全空态

```
┌─────────────────────────────┐
│     [SVG 线稿：空文件夹]     │
│     neutral-200，几何线条     │
│                             │
│   暂无记录                   │  ← h3，neutral-600
│   完成练习后，错题将自动归档至此 │  ← body-sm，neutral-400
│                             │
│   [去做题]                   │  ← Primary sm 按钮
│                             │
└─────────────────────────────┘
```

- 插图：SVG 线稿（非卡通），如空文件夹的几何线条图，neutral-200 色
- 禁止使用感叹号、鼓励性语气（"继续加油！""太棒了！"）
- 禁止使用卡通暖色系插图

### 筛选空态

```
没有符合筛选条件的错题
```
- body，neutral-400，居中
- 无插图（筛选空态不需要大块视觉）
