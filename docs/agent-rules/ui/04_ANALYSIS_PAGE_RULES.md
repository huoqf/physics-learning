## 1. 页面整体布局

```
┌──────────────────────────────────────────────┐
│  题干区（全宽，max-w-900px 居中，p-6）        │
│  bg-white，border border-neutral-200         │
│  [年份][省份][难度][知识点标签] + 题目正文    │
├──────────────────────┬───────────────────────┤
│  分步解析区（60%）    │  知识链路区（40%）    │
│  bg-white p-6        │  bg-neutral-50 p-4    │
└──────────────────────┴───────────────────────┘
```

- 题干区字号：16px，行高 1.7
- 题干区 KaTeX 公式：块级优先，行内次之
- 知识点标签：按 importance 使用 `02_UI_RULES §6` 标签色

## 2. 分步解析区

### 2.1 每步结构（手风琴）

```
[步骤编号] [步骤标题]                    [展开/折叠图标]
──── 展开后 ──────────────────────────────────────
  （可选）受力/运动分析 SVG 小图
  KaTeX 公式（块级）
  文字说明（≤ 80字）
  关联知识点标签（可点击跳转 AnimationPage）
```

### 2.2 状态样式

| 步骤状态 | 左侧竖条 | 背景 |
|---------|---------|------|
| 未到达 | neutral-200 4px | white |
| 当前步（高亮）| primary-600 4px | primary-50 |
| 已完成 | success-500 4px | white |

- 展开动效：`height + opacity`，250ms ease-in-out（见 `03_MOTION_RULES`）
- 同时展开步骤数：**≤ 2步**（防止认知过载）
- 同时展开 2 步时：第 1 步的 SVG 小图降级为缩略图（opacity 0.5 + 尺寸缩小 50%），而非完全收起，保留用户对比参考的能力
- 导航：「上一步」/「下一步」按钮，Primary sm 尺寸

### 2.3 单步展开内容上限

每步展开区内信息密度必须受控，与 Canvas "≤7 元素"精神联动：

| 内容类型 | 上限 |
|---------|------|
| SVG 小图 | 最多 1 个，元素 ≤ 5 个 |
| 公式块（KaTeX 块级） | 最多 2 个 |
| 文字说明 | ≤ 80 字 |
| 关联知识点标签 | ≤ 3 个 |

## 3. 知识链路区

### 3.1 节点样式

- 树形/线性排列：基础概念在上，综合考点在下
- 节点按 importance 显示对应标签色（参见 `02_UI_RULES §6`）
- 节点标题：**≤ 12字**，body-sm

### 3.2 联动高亮

- 当前步骤的关联节点：`box-shadow: glowRing.highlight（详见 src/theme/shadow.ts）`
- 已覆盖节点：左侧 3px success-500 色条

### 3.3 跳转行为

- 点击节点：新标签打开 `/animation/:id`（不打断当前解析进度）
- 悬停：tooltip 显示知识点一句话说明（≤ 30字）
- 步骤进度通过 URL hash 保持（`#step-N`），刷新后恢复（见 `06_NAVIGATION_RULES §2`）

## 4. 数据加载规范

- 通过 `src/data/analysisRegistry.ts` 按 id 查找题目配置
- 通过 `React.lazy + Suspense` 按需加载题目详情组件

### 4.1 骨架屏布局（与真实布局一致）

骨架屏必须反映真实页面结构，避免加载→渲染的布局跳动（CLS）。

**题干区骨架屏：**
```
├── 标签行：4 个 chip 占位（60px × 24px，间距 8px，rounded-full，bg-neutral-100）
├── 空行 16px
└── 正文：3 行文字占位
    ├── 第 1 行：100% 宽度，h-16px，bg-neutral-100，rounded
    ├── 第 2 行：90% 宽度
    └── 第 3 行：60% 宽度
```

**分步区骨架屏（手风琴折叠态）：**
```
├── 步骤 1：标题行占位（h-40px，左侧 4px 竖条 bg-neutral-200，右侧 chevron 占位）
├── 步骤 2：同上
└── 步骤 3：同上
（不展开内容，符合 §2.2「默认折叠」规则）
```

**知识链路区骨架屏：**
```
└── 6 个节点占位（圆角矩形 w-full h-36px，bg-neutral-100，间距 8px，从上到下）
```

- 骨架屏动画：`bg-neutral-100` → `bg-neutral-50` 循环闪烁，1.5s ease-in-out infinite
- 骨架屏总高度必须与真实内容首屏高度一致（避免 CLS > 0.1）

---

## 5. 禁止项

- 禁止全部步骤默认展开（加载即展开所有步骤）
- 禁止跳过步骤顺序直接显示答案（练习模式）
- 禁止知识链路区节点超过 12 个（超出时合并为父节点）
- 禁止硬编码题目内容（必须通过 analysisRegistry 加载）
